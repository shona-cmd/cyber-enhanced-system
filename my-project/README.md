# My Project

## Overview
This project is a TypeScript application that serves as a template for building web applications. It includes a structured approach to organizing controllers, routes, and types.

## Project Structure
```
my-project
├── src
│   ├── index.ts          # Entry point of the application
│   ├── controllers       # Contains controller classes
│   │   └── index.ts      # Exports IndexController
│   ├── routes            # Contains route definitions
│   │   └── index.ts      # Exports setRoutes function
│   └── types             # Contains type definitions
│       └── index.ts      # Exports Request and Response interfaces
├── package.json          # npm configuration file
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Project documentation
```

## Installation
To install the necessary dependencies, run the following command:

```
npm install
```

## Usage
To start the application, use the following command:

```
npm start
```
